Discuss the scope of var, let, and const.

1. 'var' Scope:

  • Variables declared with var are function-scoped. This means they are accessible within the function in which they are declared, regardless of where they appear within that function.

2. 'let' Scope:

  • Variables declared with let are block-scoped. This means they are only accessible within the block in which they are defined.

3. 'const' Scope:

  • Variables declared with const are also block-scoped like let. However, they have an additional characteristic: they are constants and cannot be reassigned after their initial value is set.
Tell us the use cases of null and undefined.

Use cases of null

A null value in JavaScript is used for referring absence of any object value and if any function or variable returns null, then we can infer that the object could not be created.

Use cases of undefined

In JavaScript, the value undefined is used to represent the absence of a value or the lack of a defined value for a variable or property.
